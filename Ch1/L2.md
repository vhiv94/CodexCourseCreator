# L2: Walk the Corpus with Control

## Concept

Traversal is not a detail—it is a **contract**. When the corpus is a **list** of documents in **logical order**, every pass that “walks the corpus” should see rows **from first to last** unless a lesson explicitly says otherwise. That is how you keep **summaries**, **scans**, and later **indexes** tied to the same story the list tells.

Python’s **`for`** loop visits a sequence **in order**. **`enumerate`** pairs each element with its **0-based index**—handy when you want both “which slot” and “what’s in the slot” without manual counter variables. This lesson asks you to **summarize** bodies in that order: **how many documents**, **each body’s length in list order**, and a **running notion of total characters** derived from those lengths (what the tests check is the **sum** of per-body lengths).

```python
# Illustration only: enumerate pairs each element with its 0-based index.
for position, element in enumerate(sequence):
    pass  # real code would inspect element (e.g. the text body)
```

## Why it matters in this project

Later chapters assume you can reason about **linear passes**: full scans before an inverted index, controlled iteration for previews, and predictable ordering when dicts enter the picture. If you implicitly relied on **unordered** views, summaries could **look** right on small examples yet drift from list order. L2 makes **ordered traversal** and **`enumerate`** part of your muscle memory in `corpus.py`.

## Common misconceptions

- **“Total characters should count only non-empty bodies.”** — The tests include an **empty string** body; its length is **0** but still occupies a corpus slot, and **`total_chars`** is the **sum** of **all** per-body lengths in order.
- **“I'll sort doc ids to be safe.”** — Summaries must follow **list order**, not numeric id order; the swap test encodes that.
- **“enumerate is optional style.”** — For this lesson it is **required** in **`corpus.py`** by static analysis; use it where it genuinely helps the traversal you write.
