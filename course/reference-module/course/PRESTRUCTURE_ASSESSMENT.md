# Pre-Structure Assessment

Use this short-answer check before locking the course shape.

## How to answer

- Answer in your own words.
- Short bullet answers are fine.
- If something feels fuzzy, say what you think and what still feels uncertain.

## Prompts

1. Which parts of simple Python scripts already feel familiar to you?
2. What feels less familiar: cleaning text input, filtering lists, or combining helper functions into one final output?
3. In plain language, what makes a small function easier to trust or reuse?
4. What kinds of input mistakes do you think are easiest to miss in a tiny script like this?
5. Would one focused chapter be enough for this topic, or do you expect several different concept areas?

## Routing notes

- Use the answers to confirm whether this should stay a one-chapter course or expand.
- Use the answers to decide how much review the lessons need.
- Do not skip the fundamentals entirely just because the learner already recognizes the syntax.
