# Chapter 1 Prechapter Assessment

Use this short-answer check before writing the Chapter 1 lesson files.

## How to answer

- Answer in your own words.
- Short bullet answers are fine.
- If you are unsure, include your best guess plus what still feels unclear.

## Prompts

1. How would you explain the difference between raw input and cleaned input?
2. Why might a script filter out empty notes instead of keeping every line it receives?
3. When would you split a tiny script into helper functions instead of writing everything in one block?
4. What output details would make a text summary feel predictable and easy to read?
5. Which part of this chapter would you want extra examples for before coding: name cleanup, note filtering, or final summary formatting?

## Routing notes

- Keep the lesson ids and order unchanged.
