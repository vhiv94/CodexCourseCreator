# Backend Fundamentals - Kanban Lite

This module teaches backend fundamentals through a self-hosted Kanban Lite API using FastAPI and SQLite.

## Quick start

From this directory:

```bash
uv sync
uv run pytest main_test.py -m lesson_ch1_l1
```

## Dependency management

Use uv commands for this module:

```bash
uv add <package-name>
uv remove <package-name>
```

Do not use ad-hoc `pip install` for this course workflow.
