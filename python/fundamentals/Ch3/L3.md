# Map Fields with Dictionaries

Dictionaries give names to values, which can make record access clearer than numeric indexes. In this lesson you convert tuple-shaped entry data into a dictionary with explicit keys. The tool is key selection: meaningful keys improve readability and reduce mistakes in later functions. This project step bridges positional data toward self-documenting structures. A common misconception is that dictionaries preserve "meaning" automatically; key naming quality still matters.

## Assigned task(s)

- Create `entry_to_dict(entry_tuple)` in `main.py`.
- Return a dictionary using stable keys for each field.
- Preserve all original values when mapping from tuple to dict.
