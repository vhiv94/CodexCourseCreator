# Deduplicate Categories with Sets

Sets store unique values, making them ideal for deduplication tasks. You will use a set to collect distinct categories from many tracker entries without manually checking each one. The tool focus is membership and uniqueness guarantees. This project step provides a fast way to summarize category coverage in reports. A common misconception is that set order is stable; sets are about uniqueness, not ordering.

## Assigned task(s)

- Create `unique_categories(entries)` in `main.py`.
- Return a set of category names found in the entry data.
- Ensure duplicates collapse into one value.
- **Evidence (tests):** `main_test.py` + `lesson_ch3_l4`
