# Compose Functions into Pipelines

Function composition turns several small operations into one reliable workflow. In this lesson you combine normalization, validation, and shaping into a single pipeline function. The tools are helper reuse and ordered function calls. This project step creates a reusable add-entry pipeline that main flow can call later. A common misconception is that composition hides logic; when helper names are clear, composition increases readability.

## Assigned task(s)

- Create `prepare_entry_pipeline(name, category, minutes)` in `main.py`.
- Return a single structured value that downstream layers can consume.
- Reuse existing helper behavior instead of duplicating logic.
- **Evidence (tests):** `main_test.py` + `lesson_ch4_l4`
