# Iterate Through Strings

A string is not just text; it is an iterable object that yields one character at a time. You will use a `for` loop and a simple conditional to count only alphabetic characters. This project step creates the mental bridge from strings to other iterable structures like lists and tuples. The tradeoff to notice is readability versus compact one-liners: beginner code should make loop intent obvious. A common misconception is that a string loop returns words; by default it returns characters.

## Assigned task(s)

- Create `count_alpha_characters(text)` in `main.py`.
- Iterate through the input string and count only letters.
- Return the count as an integer.
