# Index and Slice Strings

Strings are ordered sequences, so indexing and slicing let you inspect specific segments of text. In this lesson you will extract compact tag prefixes such as category labels from a single input string. The tools are indexing, slicing boundaries, and careful length assumptions. This project step is important because many CLI inputs are compact text that needs structured parsing before storage. A common misconception is that slicing includes the end index; in Python it does not.

## Assigned task(s)

- Return a substring from the start of `text` through the slice boundary.
