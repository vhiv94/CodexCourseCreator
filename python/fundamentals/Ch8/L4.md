# Complete the Personal Tracker Capstone

The capstone lesson integrates parsing, validation, persistence, and object reporting into one coherent session flow. In this lesson you implement a high-level session function that coordinates earlier components instead of replacing them. The tools are orchestration and clear boundaries between command input, data updates, and output rendering. This project step demonstrates how fundamentals combine into a working application. A common misconception is that integration means one giant function; maintain structure by delegating to existing helpers.

## Assigned task(s)

- Create `run_tracker_session(...)` in `main.py`.
- Accept command/session inputs and return a final output string.
- Integrate previously built pieces in a clear sequence.
- **Evidence (tests):** `main_test.py` + `lesson_ch8_l4`

Optional implementation sketches live in `Ch8/L4-hint.md`.
