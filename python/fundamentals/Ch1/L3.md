# Work with Basic Data Types

Python gives you multiple fundamental types because different kinds of data have different rules. Here you will practice `str`, `int`, `float`, and `bool` by building a starter profile for one user. The key tool is intentional type choice: names are text, counts are integers, fractional targets use floats, and feature toggles use booleans. This project step establishes data contracts you will rely on when entries are parsed and summarized later. A common misconception is that numbers are all the same; mixing `int` and `float` blindly can produce surprising behavior.

## Assigned task(s)

- Create `make_starter_profile(username, starting_entries, target_hours, notifications_enabled)`.
- Return a dictionary where each field preserves the expected type.
- Keep the output shape stable so later lessons can reuse it.
- **Evidence (tests):** `main_test.py` + `lesson_ch1_l3`
