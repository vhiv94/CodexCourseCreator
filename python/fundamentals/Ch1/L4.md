# Evaluate Expressions with Operators

Operators let you combine values into new values and decisions. In this lesson you will use arithmetic and comparison reasoning to compute tracker progress, which turns raw numbers into meaningful feedback. The tool focus is expression design: one clear formula is easier to test than scattered partial math. This project step introduces measurable progress so the tracker can move from storage to insight. A common misconception is that operator precedence is always intuitive; use parentheses when clarity matters more than brevity.

## Assigned task(s)

- Return a numeric percentage suitable for display.
- Handle normal non-zero targets consistently so future summaries are predictable.
