# Define a Basic Class

A class is a blueprint that groups related data and behavior. In this lesson you introduce `TrackerItem` as a dedicated model for one entry instead of passing anonymous tuples everywhere. The key tool is class declaration as a design boundary. This project step improves readability and prepares inheritance in the final chapter. A common misconception is that classes are always advanced; simple classes can clarify beginner programs quickly.

## Assigned task(s)

- Define `TrackerItem` in `main.py`.
- Ensure it is a proper class that can be instantiated later.
- Keep its purpose focused on representing one tracker entry.
- **Evidence (tests):** `main_test.py` + `lesson_ch7_l1`
