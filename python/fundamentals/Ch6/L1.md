# Read and Write Text Files

In-memory data disappears when a program exits, so persistence needs file I/O. In this lesson you save lines to disk and load them back using path-aware functions. The key tools are `Path`, text encoding, and explicit read/write boundaries. This project step gives the tracker memory across program runs. A common misconception is that file behavior is identical everywhere; paths and current working directory still matter.

## Assigned task(s)

- Create `save_entries(path, lines)` and `load_entries(path)` in `main.py`.
- Write text lines to a file and read them back in a consistent format.
- Keep file encoding explicit and stable.
- **Evidence (tests):** `main_test.py` + `lesson_ch6_l1`
