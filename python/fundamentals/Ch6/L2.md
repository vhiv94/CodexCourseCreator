# Append Records Safely

Append mode adds new records without deleting existing history. In this lesson you write one record at a time to the end of a file, preserving order. The tools are append file mode and newline discipline. This project step allows each `add` command to grow the tracker log incrementally. A common misconception is that "write" and "append" are interchangeable; write mode can erase prior data.

## Assigned task(s)

- Create `append_record(path, line)` in `main.py`.
- Append one line per call while preserving earlier lines.
- Keep output format consistent so later parsing remains simple.
