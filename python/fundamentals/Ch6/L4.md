# Validate Loaded Records

Persisted data can be malformed, so loading logic should enforce simple invariants before accepting records. In this lesson you validate line shape and reject malformed entries early. The tools are delimiter checks and structured conditionals. This project step protects the in-memory tracker model from corrupted file lines. A common misconception is that file data is always trustworthy because your own program wrote it.

## Assigned task(s)

- Create `validate_record_line(line)` in `main.py`.
- Return `True` only for lines that match your expected record shape.
- Reject malformed shapes without crashing.
- **Evidence (tests):** `main_test.py` + `lesson_ch6_l4`

Optional implementation sketches live in `Ch6/L4-hint.md`.
