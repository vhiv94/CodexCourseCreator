# Ch1/L2 - Append a Clean Note

## Concept

Clean incoming text before storing it, and reject invalid input early.

## Assigned task(s)

- Implement `append_note(notes: list[str], raw_note: str) -> list[str>` in `main.py`.
- Strip `raw_note` before storing it.
- Raise `ValueError` when the stripped note is empty.
- Return a new list with the cleaned note appended.

## Acceptance criteria

- `uv run pytest main_test.py -m lesson_ch1_l2` passes.
