# Ch1/L1 - Format a Greeting

## Concept

Write one small function that always returns the same output for the same input.

## Assigned task(s)

- Implement `format_greeting(name: str) -> str` in `main.py`.
- Strip surrounding spaces from `name`.
- Return exactly `Hello, <name>!`.

## Acceptance criteria

- `uv run pytest main_test.py -m lesson_ch1_l1` passes.
